## Стек
- Vue 3 + Vite

## Сборка

✅ Клонировать ветку "main" на локальный компьютер

```
git clone git@github.com:AnastasiiaKunstman/crypto-wallet.git

```
✅ Перейти в папку с проектом

```
cd crypto-wallet

```
✅ Установить зависимости

```
npm install

```
✅ Запустить проект

```
npm start

```
